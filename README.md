## About

백엔드와 프로덕트 관점을 함께 가져가는 **Product Engineer**입니다.  
사용자 문제를 먼저 정의하고, 제약 조건 안에서 기술적 균형점을 찾고, 운영 가능한 구조로 연결하는 과정에 집중합니다.

- **문제 정의**를 바탕으로 기능 우선순위와 제품 방향을 설계합니다.
- **데이터와 실험**으로 기술 선택의 근거를 만들고 트레이드오프를 검증합니다.
- **구현 → 배포 → 운영 → 개선**까지 이어지는 전체 사이클을 중요하게 생각합니다.
- **AI 에이전트와 워크플로우**를 실제 개발 프로세스 안에서 검증 가능한 형태로 활용합니다.
